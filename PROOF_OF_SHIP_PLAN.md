## 4. Top-10 math (sober version)

50 winners means top-10 needs excellence on **every** axis, not domination of one:
- **Tx count:** 30 real daily users × 4 tx × 15 days ≈ **1,800+ txs** — achievable with seeded users alone; spray loops push it higher
- **Unique users:** zero-deposit + welcome tickets converts casual clicks into on-chain users; target **100+ verified humans**
- **Technical quality:** already strong — protect it (tests green, README honest, no dead code)
- **Human score:** most builders phone in the video; a real-phone, real-money 4-minute demo is a visible outlier

**Biggest risks:** (1) losing days before contracts are registered — that's why Phase 0/1 is front-loaded; (2) the daily draw ever failing to resolve — keeper redundancy + manual fallback alarm; (3) farmed-looking metrics — we gate free value behind verification and report honestly (judges have flagged this before).
